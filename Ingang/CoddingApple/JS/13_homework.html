이자율 계산

첫 예금액이 5만원 미만이면, 이율이 연 15퍼센트이고,
5만원 이상이면 이율이 연 20퍼센트인 은행이 있다.
변수에 예금액을 넣으면, 2년 후의 총 예금액을 콘솔창에
출력하는 코드를 자바 스크립트를 활용하여 작성하라. 

    var 예금액 = 60000;
    var 미래예금액 = 0;

    // 여기에 코드를 넣어보세요

    console.log(미래예금액);

예상 결과
========================
입력 -> 60000, 출력 -> 86400
입력 -> 10000, 출력 -> 13225
========================
